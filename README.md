# STAT Flashcards

A lightweight flashcard website for studying statistics and probability topics from class notes.

## Files

- `index.html` - main page
- `styles.css` - site styling
- `app.js` - flashcard data and interactions

## Local use

Open `index.html` in a browser.

## Deploy

This project can be redeployed to Vercel with:

```powershell
npx vercel --yes
```
